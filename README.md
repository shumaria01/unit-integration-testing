## Project Description
This project demonstrates Unit Testing and Integration Testing using Python and pytest.
A simple banking application is tested to ensure correctness, reliability, and proper
integration between multiple functions.

## How to Run Tests
1. Install dependencies:
   pip install -r requirements.txt

2. Run all tests:
   pytest

3. Run tests with verbose output:
   pytest -v

4. Generate HTML report:
   pytest --html=report.html -v

## GitHub Actions (CI)
This repository uses GitHub Actions to automatically run unit and integration tests
whenever code is pushed or a pull request is created.
A green tick indicates successful test execution.
